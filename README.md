# :shopping_cart: Demo Web Shop Automation Framework

:rocket: This project is an end-to-end automation framework built using Selenium WebDriver and TestNG.

---

## :pushpin: Technologies Used

- :computer: Java
- :globe_with_meridians: Selenium WebDriver
- :test_tube: TestNG
- :package: Maven
- :bar_chart: Excel - Data Driven Testing

---

## :gear: Framework Features

:white_check_mark: Page Object Model (POM) design pattern  
:white_check_mark: Data Driven Testing using Excel  
:white_check_mark: Screenshot capture on test failure  
:white_check_mark: TestNG Listeners  
:white_check_mark: Reusable and scalable framework  

---

## :test_tube: Test Scenarios Covered

- :memo: User Registration (Success & Failure)
- :lock: Login 
- :shopping_bags: Add to Cart
- :credit_card: Checkout Flow

---

---

## :arrow_forward: How to Run the Project

1. Clone the repository

git clone https://github.com/vidhyavellaiyappan/demowebshop-automation.git

2. Open in Eclipse or any IDE  

3. Run `testng.xml`
